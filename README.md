# fretboarder

Generate SVG chord diagrams

A work in progress. Not even close to finished.

## About

A JavaScript library by Will McNeilly.

See the [project homepage](http://willmcneilly.github.io/fretboarder).

## Installation

Using Bower:

    bower install fretboarder

Or grab the [source](https://github.com/willmcneilly/fretboarder/dist/fretboarder.js) ([minified](https://github.com/willmcneilly/fretboarder/dist/fretboarder.min.js)).

## Usage

Basic usage is as follows:

    fretboarder();

For advanced usage, see the documentation.
